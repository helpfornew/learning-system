# 单词记忆系统

这是一个基于Web的单词记忆系统，具有每日任务和词汇地图功能。

## 功能特点

### 1. 每日任务
- 设置每日学习单词数量（5-50个）
- 系统自动从cards.json中选取单词
- 标记单词为"已掌握"或"需复习"
- 显示学习进度和统计信息

### 2. 词汇地图
- 可视化显示所有单词的掌握状态
- 颜色编码：绿色（已掌握）、橙色（需复习）、蓝色（新单词）
- 圆圈大小表示单词掌握程度
- 点击单词查看详细信息和进行标记

### 3. 数据管理
- 自动保存学习进度到浏览器本地存储
- 支持导出/导入学习数据
- 每日自动重置任务

## 使用方法

1. 直接打开 `index.html` 文件在浏览器中运行
2. 或者使用HTTP服务器运行：
   ```bash
   npx http-server -p 8080
   ```
   然后在浏览器中访问 `http://localhost:8080`

3. 首次使用：
   - 设置每日学习目标
   - 开始学习今日单词
   - 点击"已掌握"或"需复习"标记单词状态
   - 查看词汇地图了解整体进度

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 主要逻辑代码
- `cards.json` - 词汇数据文件
- `README.md` - 说明文档

## 词汇数据

系统使用 `cards.json` 文件中的词汇数据。该文件包含一个词汇数组，格式如下：
```json
{
  "vocabulary": ["word1", "word2", "word3", ...]
}
```

## 技术特点

- 纯前端实现，无需后端服务器
- 使用LocalStorage保存用户数据
- 使用D3.js实现词汇地图可视化
- 响应式设计，支持移动设备
- 中文界面，易于使用

## 浏览器支持

支持所有现代浏览器（Chrome, Firefox, Safari, Edge等）

## 注意事项

- 学习数据保存在浏览器本地，清除浏览器数据会丢失进度
- 建议定期导出数据备份
- cards.json文件可以替换为自定义的词汇列表
