# 高考学习系统 - Electron 客户端打包配置

## 两个客户端应用

本项目有两个独立的 Electron 客户端应用：

### 1. 统一学习系统 (`desktop_app/`)
- **功能**: 学习系统主页、登录界面、时间跟踪、智能提醒、数据备份
- **前端**: 静态 HTML (`config/index.html`)
- **包名**: `gaokao-learning-system_1.0.0_amd64.deb`
- **大小**: 70MB

### 2. 错题系统 (`mistake_system/mistake-system-desktop/`)
- **功能**: 错题管理、AI 分析、学习计划、复习计划
- **前端**: React + TypeScript + Vite 构建
- **包名**: `mistake-system-desktop_2.0.0_amd64.deb`
- **大小**: 78MB

## 服务器配置

两个应用都通过 `electron/config.json` 配置服务器地址：

```json
{
  "server": {
    "url": "http://192.168.1.72",
    "enableCustomServer": true,
    "allowInsecureConnection": true
  }
}
```

| 字段 | 说明 |
|------|------|
| `url` | 后端服务器地址 |
| `enableCustomServer` | 允许用户在设置中修改 |
| `allowInsecureConnection` | 允许 HTTP 非加密连接 |

## 打包步骤

### 统一配置服务器地址

```bash
# 设置服务器 IP
SERVER_IP="192.168.1.72"

# 配置统一学习系统
cat > ~/learning_system/desktop_app/electron/config.json << EOF
{
  "server": {
    "url": "http://${SERVER_IP}",
    "enableCustomServer": true,
    "allowInsecureConnection": true
  }
}
EOF

# 配置错题系统
cat > ~/learning_system/mistake_system/mistake-system-desktop/electron/config.prod.json << EOF
{
  "server": {
    "url": "http://${SERVER_IP}",
    "enableCustomServer": true,
    "allowInsecureConnection": true
  }
}
EOF
```

### 打包统一学习系统

```bash
cd ~/learning_system/desktop_app

# 直接打包（无需构建步骤）
npm run build:linux

# 输出：release/gaokao-learning-system_1.0.0_amd64.deb
```

### 打包错题系统

```bash
cd ~/learning_system/mistake_system/mistake-system-desktop

# 复制生产配置
cp electron/config.prod.json electron/config.json

# 构建并打包
npm run build:linux

# 输出：release/mistake-system-desktop_2.0.0_amd64.deb
```

## 安装包位置

| 应用 | 安装包路径 |

| 统一学习系统 | `desktop_app/release/gaokao-learning-system_1.0.0_amd64.deb` |

```bash

sudo dpkg -i ~/learning_system/desktop_app/release/gaokao-learning-system_1.0.0_amd64.deb

# 安装错题系统
sudo dpkg -i ~/learning_system/mistake_system/mistake-system-desktop/release/mistake-system-desktop_2.0.0_amd64.deb

```

## 更新服务器地址

### 方法 1：重新打包（推荐批量部署）

修改配置文件后重新打包（见上方"打包步骤"）。

### 方法 2：用户自行配置

两个客户端都内置了服务器配置界面：
- 打开客户端
- 进入「设置」页面
- 修改「服务器地址」
- 保存后重启应用

### 方法 3：修改已安装的配置文件

配置文件位置：
- Linux: `~/.config/<应用名>/config.json`

```bash
# 修改统一学习系统配置
nano ~/.config/gaokao-learning-system/config.json

# 修改错题系统配置
nano ~/.config/mistake-system-desktop/config.json
```


## 注意事项

1. **局域网访问**: 当前配置使用局域网 IP `192.168.1.72`，仅在同一局域网内可用
2. **公网访问**: 需要配置路由器端口转发（80 和 8080 端口）
3. **防火墙**: 确保客户端机器能访问服务器端口
4. **HTTP vs HTTPS**: 当前配置允许 HTTP 连接，适合内网部署
