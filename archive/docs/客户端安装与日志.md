# 高考学习系统 - 客户端打包与安装

## 两个客户端应用

| 特性 | 统一学习系统 | 错题系统 |
|------|-------------|---------|
| **目录** | `desktop_app/` | `mistake_system/mistake-system-desktop/` |
| **包名** | `gaokao-learning-system_1.0.0_amd64.deb` | `mistake-system-desktop_2.0.0_amd64.deb` |
| **大小** | 70MB | 78MB |
| **功能** | 主页、登录、时间跟踪、提醒、备份 | 错题管理、AI 分析、学习计划 |
| **前端** | 静态 HTML | React + Vite |
| **安装路径** | `/opt/GaokaoLearningSystem/` | `/opt/Mistake System Desktop/` |

## 服务器配置

两个应用都通过 `electron/config.json` 配置服务器地址：

```json
{
  "server": {
    "url": "http://192.168.1.72",
    "enableCustomServer": true,
    "allowInsecureConnection": true
  }
}
```

| 字段 | 说明 |
|------|------|
| `url` | 后端服务器地址（局域网 IP） |
| `enableCustomServer` | 允许用户在设置中修改 |
| `allowInsecureConnection` | 允许 HTTP 非加密连接 |

## 打包步骤

### 1. 统一学习系统

```bash
cd ~/learning_system/desktop_app

# 修改配置（如需要）
nano electron/config.json

# 打包
npm run build:linux

# 输出：release/gaokao-learning-system_1.0.0_amd64.deb
```

### 2. 错题系统

```bash
cd ~/learning_system/mistake_system/mistake-system-desktop

# 复制生产配置
cp electron/config.prod.json electron/config.json

# 构建并打包
npm run build:linux

# 输出：release/mistake-system-desktop_2.0.0_amd64.deb
```

## 安装与卸载

### 安装

```bash
# 安装统一学习系统
sudo dpkg -i ~/learning_system/desktop_app/release/gaokao-learning-system_1.0.0_amd64.deb

# 安装错题系统
sudo dpkg -i ~/learning_system/mistake_system/mistake-system-desktop/release/mistake-system-desktop_2.0.0_amd64.deb
```

### 卸载

```bash
# 卸载统一学习系统
sudo dpkg -r gaokao-learning-system

# 卸载错题系统
sudo dpkg -r mistake-system-desktop
```

## 端口转发配置

如需从外部网络访问，在路由器配置：
- 外部端口 80 → 内部 IP 192.168.1.72:80
- 外部端口 8080 → 内部 IP 192.168.1.72:8080

## 常见问题

### 应用无法启动

```bash
# 检查后端服务
sudo systemctl status learning-system

# 重启服务
sudo systemctl restart learning-system
```

### 连接服务器失败

1. 检查服务器地址配置
2. 确保防火墙开放端口
3. 在同一局域网内测试

### GPU 渲染问题

应用已配置禁用 GPU 加速，使用软件渲染。如需启用：

编辑 `/opt/GaokaoLearningSystem/resources/app/electron/main.js`，移除：
```javascript
app.commandLine.appendSwitch('disable-gpu')
```
