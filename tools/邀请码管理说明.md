# 邀请码管理工具

## 使用说明

### 1. 创建邀请码

```bash
# 创建随机邀请码（默认 365 天有效期，无限次使用）
python3 tools/manage_invite_codes.py create

# 创建指定代码的邀请码
python3 tools/manage_invite_codes.py create --code MYCODE2026

# 创建有限次数的邀请码（最多使用 10 次）
python3 tools/manage_invite_codes.py create --code TEMP2026 --max 10

# 创建短期邀请码（30 天有效期）
python3 tools/manage_invite_codes.py create --days 30

# 添加备注
python3 tools/manage_invite_codes.py create --remark "给高二 (3) 班的学生"
```

### 2. 查看邀请码列表

```bash
python3 tools/manage_invite_codes.py list
```

输出示例：
```
代码                 使用次数      最大次数      状态     过期时间       备注
------------------------------------------------------------------------------------------
ADMIN2026           0/无限         无限         有效     2027-03-14
INV1A2B3C4D         3/10          10          有效     2026-06-07     高二 (3) 班
------------------------------------------------------------------------------------------
共 2 个邀请码
```

### 3. 禁用/启用邀请码

```bash
# 禁用邀请码（停止使用但保留记录）
python3 tools/manage_invite_codes.py disable INV1A2B3C4D

# 重新启用邀请码
python3 tools/manage_invite_codes.py enable INV1A2B3C4D
```

### 4. 删除邀请码

```bash
python3 tools/manage_invite_codes.py delete INV1A2B3C4D
```

### 5. 环境变量

如果数据库不在默认位置，可通过环境变量指定：

```bash
export LEARNING_SYSTEM_DB=/var/lib/learning-system/database/unified_learning.db
python3 tools/manage_invite_codes.py list
```

## 邀请码状态说明

- **有效**: 未过期且已激活，可以正常使用
- **已禁用**: 管理员手动禁用，需要重新启用才能使用
- **已过期**: 超过有效期，自动失效

## 默认邀请码

系统首次启动时会自动创建默认邀请码：`ADMIN2026`
- 无限次使用
- 1 年有效期
- 用于系统管理员创建新账号
