## 快速开始

### 系统要求

- **操作系统**: Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+)
- **浏览器**: Chrome 90+, Firefox 88+, Edge 90+, Safari 14+
- **网络**: 需要互联网连接（用于用户认证）

### 启动系统

**方式一：桌面应用**

双击桌面上的 "高考学习系统" 图标启动。

**方式二：浏览器访问**

打开浏览器，访问：`http://localhost:8080`

## 用户注册与登录

### 注册账号

1. 点击登录页面的 "注册账号" 按钮
2. 输入用户名（建议使用真实姓名或昵称）
3. 设置密码（建议包含字母和数字，至少6位）
4. 输入邀请码（向已注册用户索取）
5. 点击 "注册" 完成

> **提示**: 注册成功后自动获得 30 天 VIP 权限。

### 登录系统

1. 输入用户名和密码
2. 点击 "登录"
3. 登录成功后会自动跳转到主页

### 邀请好友

VIP 用户可以创建邀请码：

1. 进入 "设置" 页面
2. 点击 "生成邀请码"
3. 设置使用次数（0 表示无限次）
4. 将邀请码分享给好友

## 错题管理

### 添加错题

**方式一：手动录入**

1. 点击左侧菜单 "错题本"
2. 点击右上角 "+ 添加错题" 按钮
3. 填写错题信息：
   - **科目**: 选择数学/物理/化学/语文/英语/政治
   - **题目内容**: 输入题目描述
   - **错误答案**: 输入当时的错误答案
   - **正确答案**: 输入正确答案
   - **错误原因**: 分析错误原因
   - **知识点**: 添加相关知识点标签
   - **难度**: 选择 1-5 级难度
4. 点击 "保存"

**方式二：拍照上传**

1. 点击 "添加错题"
2. 点击 "拍照上传" 区域
3. 拍摄或上传题目照片
4. 系统自动识别题目内容
5. 补充其他信息后保存

### 查看错题

1. 在错题列表页面，可以看到所有错题
2. 使用筛选器按科目、难度、标签筛选
3. 点击错题卡片查看详情
4. 支持搜索题目内容

### 编辑错题

1. 点击错题卡片进入详情页
2. 点击 "编辑" 按钮
3. 修改需要更新的字段
4. 点击 "保存"

### 删除错题

1. 在错题详情页点击 "删除" 按钮
2. 确认删除操作

> **注意**: 删除操作不可恢复，请谨慎操作。

### 复习模式

1. 点击 "开始复习" 按钮
2. 系统根据艾宾浩斯遗忘曲线推荐需要复习的错题
3. 隐藏答案，尝试重新解答
4. 查看正确答案和解析
5. 标记掌握程度：
   - ✅ 已掌握：不再需要复习
   - ❌ 未掌握：加入下次复习队列

## 单词本

### 添加单词

1. 点击左侧菜单 "单词本"
2. 点击 "+ 添加单词"
3. 填写单词信息：
   - **单词**: 英文单词
   - **音标**: 可选，自动补充
   - **释义**: 中文翻译
   - **例句**: 可选，帮助记忆
   - **词性**: 名词/动词/形容词等
   - **标签**: 如 "高考核心词"、"易错词"
4. 点击 "保存"

### 批量导入

1. 在单词本页面点击 "批量导入"
2. 选择导入方式：
   - **粘贴文本**: 每行一个单词，格式：`单词 释义`
   - **上传文件**: 支持 CSV 格式
3. 预览导入内容
4. 点击 "确认导入"

**CSV 格式示例**:
```csv
word,phonetic,definition,tags
apple,/ˈæp.l/,苹果,水果
book,/bʊk/,书,日常
```

### 单词复习

系统使用艾宾浩斯遗忘曲线算法安排复习：

1. 点击 "开始复习"
2. 显示单词，回忆释义
3. 点击 "显示答案"
4. 自我评估：
   - **认识**: 延长下次复习间隔
   - **模糊**: 保持当前间隔
   - **忘记**: 缩短复习间隔，增加复习频率

### 单词统计

- **总词汇量**: 已添加的单词总数
- **已掌握**: 熟练度达到 100% 的单词
- **学习中**: 正在复习的单词
- **新单词**: 还未开始复习的单词

## 学习时间表

### 查看时间表

1. 点击左侧菜单 "时间表"
2. 查看今日学习计划
3. 点击日期切换查看其他日期

### 自定义时间表

1. 点击 "编辑时间表"
2. 选择要修改的日期（周一到周日）
3. 添加学习时段：
   - **开始时间**: 如 09:00
   - **结束时间**: 如 10:30
   - **学习内容**: 如 "数学 - 函数"
   - **标签**: 学习/休息/锻炼
4. 拖拽调整时段顺序
5. 点击 "保存"

### 时间表模板

系统提供默认时间表模板：

**单日（周一/三/五）**:
- 09:00-10:30: 化学
- 10:50-12:00: 数学
- 14:00-15:00: AI 编程项目
- 15:10-16:30: 数学错题复盘
- 17:00-18:30: 政治
- 20:00-21:00: 化学反刍
- 21:15-22:00: 英语单词

**双日（周二/四/六）**:
- 09:00-10:30: 化学
- 10:50-12:00: 物理
- 15:10-16:30: 物理错题复盘
- 17:00-18:30: 语文

**周日**: 周测日

## 学习时间跟踪

### 开始计时

1. 点击左侧菜单 "学习时间"
2. 选择科目
3. 点击 "开始计时" 按钮
4. 学习结束后点击 "停止"

### 手动添加记录

1. 点击 "添加记录"
2. 选择日期
3. 选择科目
4. 输入学习时长（分钟）
5. 点击 "保存"

### 学习统计

- **今日学习**: 当天的学习时长
- **本周学习**: 本周累计时长
- **本月学习**: 本月累计时长
- **学科分布**: 各科目的学习时长饼图

### 目标设置

1. 进入 "设置" 页面
2. 设置每日目标学习时长（建议 8-10 小时）
3. 设置高考倒计时目标日期
4. 系统会显示完成进度

## 学习平台

### 浏览科目

1. 点击左侧菜单 "学习平台"
2. 选择科目：
   - ⚗️ 化学
   - ⚡ 物理
   - 📖 语文
   - 🔢 数学
   - 🔤 英语
   - 🏛️ 政治
3. 查看科目章节列表

### 学习章节

1. 点击章节标题进入学习
2. 阅读章节内容
3. 使用涂鸦功能做笔记：
   - 选择画笔颜色
   - 在内容上标注重点
   - 保存涂鸦
4. 标记章节为 "已掌握"

### 阅读进度

- 章节列表显示阅读状态
- 灰色：未阅读
- 蓝色：已阅读
- 绿色：已掌握
- 进度条显示完成百分比

### 涂鸦功能

1. 在章节页面点击 "涂鸦模式"
2. 选择画笔工具：
   - 画笔：自由绘制
   - 荧光笔：标记重点
   - 橡皮擦：擦除
3. 选择颜色
4. 在画布上绘制
5. 点击 "保存" 或 "清除"

## AI 分析

### 配置 AI 服务

1. 进入 "设置" 页面
2. 点击 "AI 配置"
3. 选择 AI 提供商：
   - DeepSeek
   - 通义千问
   - OpenAI（需要 API Key）
4. 输入 API Key（可选）
5. 点击 "保存"

### 错题 AI 分析

1. 在错题详情页点击 "AI 分析"
2. 系统调用 AI 服务：
   - 分析错误原因
   - 提供详细解析
   - 推荐相关知识点
   - 生成类似练习题
3. 查看 AI 分析结果
4. 可选择保存分析结果

### 周学习分析

1. 点击 "AI 分析" 菜单
2. 选择 "周分析报告"
3. 系统生成包含以下内容的报告：
   - 本周错题统计
   - 薄弱知识点分析
   - 学习建议
   - 下周学习重点推荐

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**

A: 目前系统不支持自助找回密码，请联系管理员重置密码。

**Q: 邀请码在哪里获取？**

A: 向已注册的 VIP 用户索取邀请码，每个 VIP 用户可以生成多个邀请码。

### 功能问题

**Q: 错题可以导出吗？**

A: 是的，在错题列表页面点击 "导出" 按钮，支持导出为 PDF 或 Word 格式。

**Q: 支持多设备同步吗？**

A: 支持。只要使用同一账号登录，数据会自动同步到云端。

**Q: AI 分析需要付费吗？**

A: 基础 AI 分析功能免费。如需使用自己的 API Key 调用高级模型，需自行承担 API 费用。

### 技术问题

**Q: 系统无法启动怎么办？**

A: 请尝试以下步骤：
1. 检查端口 8080 是否被占用
2. 查看日志文件 `logs/server.log`
3. 重启电脑后重试
4. 联系技术支持

**Q: 数据会丢失吗？**

A: 系统会定期自动备份数据到 `data/backup` 目录。建议定期手动导出备份。

**Q: 支持离线使用吗？**

A: 部分功能支持离线使用，但用户认证、数据同步、AI 分析等功能需要联网。

### 学习建议

**Q: 如何高效使用错题本？**

A:
1. 当天错题当天录入
2. 定期使用复习功能（建议每周至少 2 次）
3. 认真填写错误原因和知识点
4. 对重点错题添加醒目标签

**Q: 单词记忆有什么技巧？**

A:
1. 利用碎片时间复习（建议每天 3 次，每次 10 分钟）
2. 结合例句记忆
3. 对易混词添加对比标签
4. 坚持每日打卡
